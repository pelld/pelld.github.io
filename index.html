Capping values in R

Five-a-day

When calculating the five-a-day value for an individual smoothie and fruit juice consumption, along with beans, should be capped at a certain value. That is regardless of how many grams of these foods you eat per day they only count as a limited amount towards your five-a-day value.

This can be coded in r using the pmin() function this takes two arguments, firstly the value to be capped eg. NDNS$beansg and the amount to cap at eg 150g which gives:

pmin(NDNS$beansg, 150)
